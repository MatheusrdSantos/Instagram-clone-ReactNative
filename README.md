# Instagram-clone-ReactNative
Instagram-based interface implemented with ReactNative

## Functionalities
* List posts
* Like a post
* Create a new post
* Realtime feed (see this repo: [Instagram-API-NodeJs](https://github.com/MatheusrdSantos/Instagram-API-Nodejs))
